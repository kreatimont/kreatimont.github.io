Политика конфиденциальности

Применяется к: My Clients

Дата вступления в силу: 5 февраля 2025 г.

Настоящая Политика конфиденциальности описывает, как собираются, используются и защищаются ваши персональные данные при использовании приложения «My Clients». Используя данное приложение, вы соглашаетесь с условиями настоящей Политики.

1. Сбор данных

«My Clients» — это приложение для управления записями, которое хранит данные локально на вашем устройстве. Все данные о записях хранятся на устройстве и не отправляются на внешние серверы. При этом приложение предоставляет функцию резервного копирования данных через iCloud.

Резервное копирование iCloud

Если вы выбираете использование резервного копирования iCloud, ваши данные могут храниться через сервис Apple iCloud. Резервные копии iCloud зашифрованы и хранятся Apple в соответствии с их политикой конфиденциальности.

Сторонние SDK

«My Clients» включает следующие сторонние сервисы:

  • Applovin SDK: Используется для показа рекламы в приложении. Может собирать определённые данные для персонализации рекламы. Подробнее — в Политике конфиденциальности Applovin.
  • Google AdMob SDK: Используется для показа рекламы в приложении. Может собирать персональные данные для таргетинга. Подробнее — в Политике конфиденциальности Google.
  • Firebase Analytics: Используется для аналитики, чтобы понимать, как пользователи взаимодействуют с приложением. Firebase Analytics не собирает персональные данные. Подробнее — в Политике конфиденциальности Firebase.
  • RevenueCat SDK: Используется для управления подписками и покупками. Может собирать данные о покупках. Подробнее — в Политике конфиденциальности RevenueCat.

2. Использование данных

Мы не собираем персональную информацию без вашего явного согласия (например, через резервное копирование iCloud). Приложение использует сторонние SDK для показа рекламы и аналитики. Эти SDK могут собирать данные для персонализированной рекламы, аналитики и покупок в приложении.

Цели сбора данных

  • Для показа рекламы в приложении (через Applovin SDK и Google AdMob SDK).
  • Для отслеживания использования приложения в аналитических целях (через Firebase Analytics).
  • Для обработки покупок и подписок (через RevenueCat SDK).
  • Для резервного копирования данных в iCloud (если включено пользователем).

3. Передача данных

Мы не передаём ваши персональные данные третьим лицам, кроме случаев, предусмотренных законом или описанных в настоящей политике в отношении сторонних SDK. Сторонние сервисы, интегрированные в приложение, могут собирать и использовать данные в соответствии со своими политиками конфиденциальности.

4. Хранение данных

Мы не храним персональные данные в приложении. Если вы выбираете резервное копирование через iCloud, данные хранятся в соответствии с политикой хранения данных Apple iCloud. Сторонние SDK могут хранить данные по мере необходимости для своих сервисов.

5. Ваши права

Вы можете управлять своими данными и настройками конфиденциальности:

  • Отключив резервное копирование iCloud, если не хотите хранить данные в облаке Apple.
  • Изменив настройки конфиденциальности устройства для управления персонализацией рекламы.
  • Отключив или ограничив использование сторонних SDK в настройках устройства.

6. Безопасность

Мы принимаем разумные меры для защиты ваших данных. Однако ни один способ передачи или хранения данных не является абсолютно безопасным. Мы не можем гарантировать полную безопасность передаваемых данных.

7. Изменения в политике

Мы можем обновлять настоящую Политику конфиденциальности. При обновлении мы публикуем новую версию на этой странице с обновлённой датой. Рекомендуем периодически проверять политику.
