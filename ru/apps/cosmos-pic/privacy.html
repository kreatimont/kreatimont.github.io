Политика конфиденциальности

Применяется к: Cosmos Pic

Последнее обновление: март 2025 г.

Обзор

Cosmos Pic («Приложение») разработано и поддерживается Александром Надтокой («я», «меня», «мой»). Настоящая Политика конфиденциальности объясняет, как Приложение обрабатывает вашу информацию.

Cosmos Pic не собирает, не хранит и не передаёт никаких персональных данных. Для использования Приложения не требуется аккаунт, вход или персональные данные.

Данные, к которым обращается Приложение

Приложение получает общедоступные изображения и текст из API NASA «Астрономическая картинка дня» (APOD). Это публичный API только для чтения — никакие персональные данные не отправляются в NASA в рамках этого запроса, кроме стандартных данных HTTPS-запроса (например, IP-адреса, обрабатываемого инфраструктурой NASA согласно Политике конфиденциальности NASA).

Что мы не собираем

  • Персональные идентификационные данные
  • Данные аккаунта или входа
  • Данные о геолокации
  • Аналитику или данные отслеживания поведения
  • Идентификаторы устройства

Сохранённые снимки

Если вы сохраняете снимок в фотобиблиотеку, это действие выполняется локально на вашем устройстве с использованием стандартных функций iOS. Приложение не загружает и не хранит сохранённые снимки где-либо ещё.

Безопасность данных

Поскольку персональные данные не собираются и не передаются, угрозы их утечки нет. Все сетевые запросы осуществляются по протоколу HTTPS.

Конфиденциальность детей

Cosmos Pic не собирает персональных данных ни от одного пользователя, включая детей. Приложение подходит для всех возрастов.

Изменения в политике

Настоящая Политика конфиденциальности может обновляться. Изменения публикуются на этой странице с новой датой. Продолжение использования Приложения означает принятие обновлённой политики.
